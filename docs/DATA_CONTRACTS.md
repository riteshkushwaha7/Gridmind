# GRIDMIND — Data Contracts

All routers live inside the **single** `data-nodes` FastAPI app at port **8001**. The app
exposes one shared `GET /health` plus the per-node endpoints below. Every node also
publishes to its Redis Stream and mirrors the latest payload to `latest:<stream>`.

## node-ocpp (`/ocpp`)

| Field          | Value                                                                                                                                                                                                                                                       |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints      | `GET /ocpp/status/{zone_id}`, `GET /ocpp/sessions/active/{zone_id}`, `GET /ocpp/sessions/history?zone_id=&hours=`                                                                                                                                          |
| Event payload  | `{charger_id, zone_id, node_id:"ocpp", timestamp, event_type, connector_id, id_tag, meter_start_wh, meter_stop_wh, timestamp_start, timestamp_stop, duration_minutes, energy_delivered_kwh, power_kw, connector_status, soc_start_pct, soc_end_pct, charger_type, ocpp_version, error_code}` |
| Event types    | `StartTransaction`, `StopTransaction`, `MeterValues`, `StatusNotification`                                                                                                                                                                                  |
| Redis stream   | `ocpp_events`                                                                                                                                                                                                                                               |
| Frequency      | Event-driven; simulator tick = 30 s                                                                                                                                                                                                                         |

## node-grid (`/grid`)

| Field          | Value                                                                                                                                                                                                                                                                |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints      | `GET /grid/feeder/{zone_id}/status`, `GET /grid/feeder/{zone_id}/history?hours=`, `GET /grid/feeders/summary`                                                                                                                                                         |
| Response       | `{zone_id, feeder_id, node_id:"grid", timestamp, feeder_capacity_kw, transformer_capacity_kva, transformer_limit_kw, base_load_kw, ev_load_kw, total_load_kw, headroom_kw, headroom_pct, load_factor, power_factor, voltage_pu, frequency_hz, active_evs, feeder_status, constraint_flag, peak_today_kw, demand_response_active}` |
| Redis stream   | `grid_telemetry`                                                                                                                                                                                                                                                     |
| Frequency      | 15 minutes                                                                                                                                                                                                                                                            |

## node-solar (`/solar`)

| Field          | Value                                                                                                                                                                                                                                                                                                |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints      | `GET /solar/{zone_id}/current`, `GET /solar/{zone_id}/forecast?hours=`, `GET /solar/summary`                                                                                                                                                                          |
| Response       | `{zone_id, node_id:"solar", timestamp, pv_capacity_kw, irradiance_wm2, temperature_c, cloud_cover_pct, pv_output_kw, pv_efficiency_pct, battery_soc_pct, battery_capacity_kwh, battery_charge_kw, grid_export_kw, self_consumption_kw, forecast_generation_kwh, sunrise, sunset, day_length_hours}` |
| Redis stream   | `solar_generation`                                                                                                                                                                                                                                                  |
| Frequency      | 15 minutes (reads `cloud_cover_pct` from latest weather payload)                                                                                                                                                                                                       |

## node-tariff (`/tariff`)

| Field          | Value                                                                                                                                                                                                                                                                                                                  |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints      | `GET /tariff/current`, `GET /tariff/forecast?hours=`, `GET /tariff/history?hours=`                                                                                                                                                                                                                                       |
| Response       | `{zone_id:"ALL", node_id:"tariff", timestamp, pricing_mode:"ToU", tier, rate_inr_per_kwh, next_tier, next_tier_change_minutes, demand_charge_inr_per_kw, tou_schedule:{off_peak,mid_peak,peak,super_peak}, rtp_premium_pct, ev_incentive_active, ev_incentive_rate_inr, grid_stress_signal}`                            |
| Redis stream   | `tariff_signals`                                                                                                                                                                                                                                                                                                       |
| Frequency      | On tier change + 15-min heartbeat                                                                                                                                                                                                                                                                                       |

## node-ev-session (`/ev`)

| Field          | Value                                                                                                                                                                                                                                                                                                                                                                              |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints      | `GET /ev/zones/snapshot`, `GET /ev/zone/{zone_id}/metrics?hours=`, `GET /ev/fleet/summary`                                                                                                                                                                                                                                                                                          |
| Response       | `{zone_id, node_id:"ev_session", timestamp, active_sessions, queued_evs, available_chargers, total_chargers, utilization_pct, avg_soc_arriving, avg_soc_departing, avg_session_duration_min, avg_energy_per_session_kwh, total_energy_delivered_kwh, demand_kwh_15min, demand_kwh_1h, demand_kwh_24h, ev_types:{two_wheeler_pct,three_wheeler_pct,four_wheeler_pct,bus_pct}, peak_hour_today, demand_forecast_1h_kwh}` |
| Redis stream   | `ev_analytics`                                                                                                                                                                                                                                                                                                                                                                      |
| Frequency      | 15 minutes (aggregates over OCPP active sessions + SQLite history)                                                                                                                                                                                                                                                                                                                  |

## node-weather (`/weather`)

| Field          | Value                                                                                                                                                                                                                                                                                                                            |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints      | `GET /weather/current`, `GET /weather/forecast?hours=`                                                                                                                                                                                                                                                                            |
| Response       | `{zone_id:"ALL", node_id:"weather", timestamp, temperature_c, feels_like_c, humidity_pct, wind_speed_kmh, precipitation_mm, cloud_cover_pct, visibility_km, uv_index, direct_radiation_wm2, diffuse_radiation_wm2, global_radiation_wm2, is_monsoon, weather_condition, ev_demand_modifier, solar_generation_modifier, grid_stress_weather}` |
| Redis stream   | `weather_data`                                                                                                                                                                                                                                                                                                                  |
| Source         | Open-Meteo `current=` endpoint (free, no key) with synthetic fallback when unreachable                                                                                                                                                                                                                                            |
| Frequency      | 30 minutes                                                                                                                                                                                                                                                                                                                        |
